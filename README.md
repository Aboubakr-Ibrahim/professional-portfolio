# Aboubakr Biomedical — BioOS v1.0

Production portfolio platform for Aboubakr Ibrahim, a biomedical engineer connecting clinical systems, medical imaging, medical-device quality, biomedical signals, software, and responsible healthcare-data research.

## v1.0 modules

- Professional identity and education
- Clinical engineering and radiology internships
- Privacy-screened field evidence
- Five public engineering repositories
- Medical-device quality and ISO 13485 pathway
- Seventeen selected credentials
- Two academic recommendations
- One main public CV and three role-aligned CV pathways
- International mobility and collaboration center

The platform is organized as a modular biomedical operating system. `platform-manifest.json` defines the release contract so v1.1, v2.0, and future chapters can extend the platform without replacing its evidence base.

## Local preview

Open `index.html` directly or run a simple local server:

```bash
python -m http.server 4173
```

Then visit `http://localhost:4173`.

## Privacy and evidence

- Internship claims are based on documented reports and use supervised-contribution language.
- No patient-level data, hospital records, device serial numbers, or confidential service documents are included.
- Certificate images are displayed as issued.
- Research repositories distinguish implementation evidence from unverified performance.
- The hero artwork is a branded visual composition and is not evidence of clinical deployment.

## Deployment

The site is static and can be deployed directly to Vercel or GitHub Pages without a build step.
